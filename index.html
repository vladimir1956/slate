Возможности

Большинство календарей предназначены для команд. Slate предназначен для
фрилансеров, которым нужен удобный способ планировать свой график.

  • Полезные подсказки

    Когда вы добавите
    задачу в свой календарь,
    Slate предложит полезную
    информацию.

  • Умный интерфейс

    Интерфейс приложения
    автоматически подстраивается
    под вашу текущую задачу.

  • Синхронизация

    Slate синхронизируется
    со всеми вашими
    календарями.
